Usage

Options

  • maxage: default to 0.

  • hidden: Allow transfer of hidden files, detault to false.

  • index: Default file name, detault to ‘index.html’.

  • defer: If true, serves after return next().

  • gzip: default to true.

  • extensions: default to false.

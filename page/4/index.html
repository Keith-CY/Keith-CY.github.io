    JavaScript 事件与性能

    2016-09-27

    事件处理程序本质上是一种函数, 是一种对象, 存放在内存中, 设置大量的事件处理程序会使内存中的对象变多, Web 程序的性能会变得越来越差.

    为了更好利用事件处理程序, 出现了事件委托, 用来提升性能

    事件委托

    Event Delegate: 把若干个子节点上的相同事件的处理函数绑定到父节点, 从父节点统一处理子节点冒泡上来的事件, 这种技术叫做事件委托.

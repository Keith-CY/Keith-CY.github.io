RegExp Attribtues

  • g: global
  • i: ignoreCase
  • m: multi-lines
  • source: literal text
  • lastIndex: mark the index where to start next match

Example

var re = new RegExp('^[0-9]+$','g');
console.log(re.source); //^[0-9]+$
